# Eclipsed Horizon Companion

A comprehensive companion tool for creating custom TTRPG universes using Cepheus and Traveller rulesets.

## Features
- Star System Generator
- Sophont Builder  
- Procedural History Generator
- NPC Generator
- Family Tree Viewer

## Status
Under active development - Phase 0, Sprint 1

```bash
npm install
npm run dev
