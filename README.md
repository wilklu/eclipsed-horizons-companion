# Eclipsed Horizons Companion

Integrated Traveller 2nd Edition tools and house rules documentation.

## Quick Links

- 📚 [Read the Wiki](https://wilklu.github.io/eclipsed-horizons-companion/)
- 🎮 **Applications** - Sophont Generator, World Builder, Character Generator
- 📋 **House Rules** - Centralized rule management

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run local preview: `mkdocs serve`
3. View at `http://localhost:8000`
