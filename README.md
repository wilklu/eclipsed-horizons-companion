# Eclipsed Horizon Companion

A comprehensive companion tool for creating custom TTRPG universes, independent of existing canons.

## Vision

Generate entire galaxies with procedurally created star systems, intelligent species (sophonts), fauna, NPCs, and procedural history for civilizations.

## Features

### Phase 1 (Current)

- ⭐ **Star System Generator** - Generate stars and worlds based on World Builders Handbook
- 🧬 **Sophont Generator** - Create intelligent species with cultures and histories
- 🎭 **NPC Generator** - Generate non-player characters

### Phase 2 (Planned)

- 🦁 **Beast Maker** - Generate fauna for planets
- 📜 **Procedural History** - Create civilization timelines
- 👨‍👩‍👧 **Family Tree Viewer** - Track lineage across generations

### Phase 3 (Planned)

- 🚀 **Spaceship Generator** - Design and build vessels
- 🗺️ **Galaxy Map Viewer** - Navigate star systems
- 📖 **Encyclopedia/Wiki** - Database of game information

## Tech Stack

- **Frontend:** Vue 3 + Vite
- **Backend:** Node.js + Express
- **Database:** SQLite
- **Development:** Visual Studio Code + Git/GitHub

## Getting Started

### Prerequisites

- Node.js 18+ (https://nodejs.org/)
- Visual Studio Code (https://code.visualstudio.com/)
- Git (https://git-scm.com/)

### Installation

```bash
git clone https://github.com/YOUR-USERNAME/eclipsed-horizon.git
cd eclipsed-horizon

# Backend setup (Backend runs on localhost:3000)
cd backend
npm install
npm run dev

#Frontend setup, In another terminal (Frontend runs on localhost:5173)
cd frontend
npm install
npm run dev
```
