# Quick Start Guide

Welcome! This guide will get you up and running quickly.

## 5-Minute Setup

### System Generator (Vue 3 Frontend)

```bash
cd apps/system-generator/frontend
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.

### Documentation Site

```bash
npm install          # from repo root
npm run docs:dev     # preview at http://localhost:5173
```

## Your First Universe

1. **Galaxy Survey** — Click "Create New Galaxy", enter a name and type, then click Create.
2. **Sector Survey** — Select your galaxy and click "Class 0 Sector Survey →". Pick a density and generate.
3. **Star System Builder** — Click a hex with a star, then "Class I Stellar Survey →" to detail the system.
4. **World Builder** — For habitable worlds, continue to the World Builder (Class II–IV) to generate a full UWP profile.
5. **Sophont Generator** — Use the nav bar to create intelligent species that inhabit your worlds.
6. **History Generator** — Generate a procedural history for any civilization.

All data persists in your browser's localStorage between sessions.
