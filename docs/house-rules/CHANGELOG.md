# House Rules Changelog

All notable changes to the Eclipsed Horizons house rules are documented here.

Format: `[VERSION] YYYY-MM-DD — Rule ID — Change Description`

---

## [1.3] 2024-03-01

### Added
- **HR-TC-001** (Trade & Commerce) — Extended Purchase Negotiation: brokers may research market before buying

### Changed
- **HR-CC-002** (Character Creation) — Background Skills Cap reduced from 4 to 3 skills after playtesting showed 4 was too powerful

---

## [1.2] 2024-02-20

### Added
- **HR-ST-001** (Space Travel) — Misjump Severity table combined from Classic and Mongoose rules
- **HR-ST-002** (Space Travel) — Jump masking rules with gas giant variant
- **HR-ST-020** (Space Travel) — Piracy encounter reaction table for Eclipsed Horizons subsectors

### Changed
- **HR-CO-011** (Combat) — Suppressive Fire reworked; previous version only affected movement, now also pins targets

---

## [1.1] 2024-02-08

### Added
- **HR-CO-001** (Combat) — Side-based initiative adopted
- **HR-CO-002** (Combat) — Tactical movement dash penalty
- **HR-CO-010** (Combat) — Called Shots (optional)
- **HR-CO-020** (Combat) — Grappling rules

### Experimental (Under Playtesting)
- **HR-CO-030** (Combat) — Heroic Last Stand (started playtesting Session 8)
